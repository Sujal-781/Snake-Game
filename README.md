# Snake Game 🐍

A simple Snake game built using Java and Swing.

## 🎮 Features
- Classic snake movement
- Score tracking
- Game over screen
- Random apple placement

## 🛠️ Technologies Used
- Java
- Java Swing for GUI
- IntelliJ IDEA

## 🚀 How to Run
1. Clone this repository:
   git clone https://github.com/Sujal-781/Snake-Game.git
2. Open the project in IntelliJ IDEA.
3. Run `Main.java`.
